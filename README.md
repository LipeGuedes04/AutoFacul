# 🚗 Cadastro de Veículos em Java

## 📌 Descrição

Este projeto consiste em uma aplicação em Java que permite realizar o cadastro de um veículo, considerando informações básicas como tipo, modelo, cor e massa.

O sistema diferencia automaticamente entre automóveis e motos, solicitando informações específicas para cada tipo, como quantidade de portas ou rodas.

---

## 🚀 Funcionalidades

* Seleção do tipo de veículo:

  * Automóvel
  * Moto
* Cadastro de informações básicas:

  * Modelo
  * Cor
  * Massa
* Entrada de dados específicos:

  * Número de portas (automóveis)
  * Número de rodas (motos)
* Exibição formatada dos dados do veículo

---

## 🧱 Estrutura do Projeto

O projeto é dividido em duas classes principais:

* `automovel` → Classe principal responsável pela execução do programa e interação com o usuário
* `auto1` → Classe responsável por armazenar e organizar os dados do veículo

---

## 🖥️ Tecnologias Utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Entrada de dados com `Scanner`

---

## 📚 Conceitos Aplicados

* Entrada e saída de dados
